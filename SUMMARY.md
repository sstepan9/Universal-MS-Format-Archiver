## 🔐 Безопасность

### Криптография

```
┌─────────────────────────────────────────────────────┐
│ Алгоритм:  AES-256-CBC                              │
│ Ключ:      256 бит (32 байта)                       │
│ IV:        128 бит (16 байт) — случайный           │
│ Соль:      128 бит (16 байт) — случайная           │
│ KDF:       PBKDF2HMAC (SHA-256, 100,000 итераций)   │
│ Padding:   PKCS7                                    │
└─────────────────────────────────────────────────────┘
```

### Защита

- ✅ Невозможно угадать ключ без пароля
- ✅ Каждый архив имеет уникальную соль
- ✅ 100,000 итераций PBKDF2 против атак перебором
- ✅ Стандарт NIST (неуязвим при текущих технологиях)

## 📋 Быстрый старт

### Установка

```bash
cd UMSFA
pip install -r requirements.txt
```

### Использование

```bash
# Архивирование с паролем
python main.py create my_folder backup.ms -p "SecurePassword123"

# Информация об архиве
python main.py info backup.ms

# Распаковка
python main.py extract backup.ms extracted -p "SecurePassword123"
```

### Использование в Python

```python
from umsfa import MSArchiver

archiver = MSArchiver(password="MyPassword")
archiver.archive("my_folder", "backup.ms")
archiver.extract("backup.ms", "restored")
```

```bash
python main.py create "C:\Documents" documents_backup.ms -p "VerySecurePassword!"
```

### Пример 2: Безопасная отправка файлов

```bash
python main.py create project_files project.ms -i
# Отправить project.ms по защищённому каналу
# Получатель:
python main.py extract project.ms extracted_project -i
```

### Пример 3: Автоматическое резервное копирование

```python
from umsfa import MSArchiver
import os
from datetime import datetime

def daily_backup(folder, password):
    timestamp = datetime.now().strftime("%Y%m%d")
    output = f"backup_{timestamp}.ms"
    
    archiver = MSArchiver(password=password)
    if archiver.archive(folder, output):
        print(f"✅ Резервная копия создана: {output}")
```

## 🌟 Особенности UMSFA

### Преимущества

✅ Использует современное AES-256 шифрование  
✅ Безопасное получение ключа через PBKDF2  
✅ Новый формат .ms с собственной спецификацией  
✅ Легко встраивается в Python код  
✅ Полная документация на русском языке  
✅ Готовые примеры для использования  
✅ Кроссплатформенность  
✅ MIT лицензия  

### Недостатки (по сравнению с 7zip, WinRAR)

❌ Нет поддержки распространённых форматов (ZIP, RAR)  
❌ Нет встроенной проверки целостности  
❌ Медленнее специализированных архиваторов  
❌ Нет GUI интерфейса  

**Но UMSFA компенсирует это встроенным шифрованием и простотой использования!**
