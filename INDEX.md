# 📖 INDEX — Справочник UMSFA

## 🎯 О проекте

**Universal MS Format Archiver (UMSFA)** — это мощный локальный архиватор для безопасного хранения данных с использованием **AES-256 шифрования** в новом формате `.ms`.

- **Версия:** 1.0
- **Язык:** Python 3.7+

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Установка
```bash
cd UMSFA
pip install -r requirements.txt
```

### Шаг 2: Архивирование
```bash
# С паролем (AES-256)
python main.py create my_folder backup.ms -p "SecurePassword123!"

# Без пароля
python main.py create my_folder backup.ms
```

### Шаг 3: Распаковка
```bash
python main.py extract backup.ms restored_folder -p "SecurePassword123!"
```

## 🔐 Безопасность

### Использованные технологии

- ✅ **AES-256-CBC** — Государственный стандарт шифрования
- ✅ **PBKDF2HMAC** — Безопасное получение ключа из пароля

- ✅ **Случайная соль** — 128-битная уникальная соль для каждого архива

## ❓ Часто задаваемые вопросы

### Общие вопросы

**Q: Какой формат использует UMSFA?**  
A: Собственный формат `.ms` (MS = MS Archive) с сигнатурой `UMSFA1.0`.

**Q: Мой архив совместим с другими архиваторами?**  
A: Нет. UMSFA — это независимый формат. Используйте UMSFA для распаковки.

**Q: Могу ли я открыть .ms файл на другой машине?**  
A: Да, если там установлены Python 3.7+ и зависимости из `requirements.txt`.

### Безопасность

**Q: Что произойдет, если я потеряю пароль?**  
A: Архив станет недоступным. Пароль невозможно восстановить. Храните пароль надежно!

**Q: Насколько безопасен AES-256?**  
A: Это государственный стандарт NIST. При текущих технологиях является неуязвимым.

**Q: Могу ли я присоединить цифровую подпись?**  
A: В версии 1.0 нет. Запланировано для версии 2.0.

### Техничекские

**Q: Какой максимальный размер архива?**  
A: До 2 ТБ (ограничение файловой системы NTFS/ext4).

**Q: Поддерживает ли UMSFA многопоточность?**  
A: Частично. tar.gz обработка использует многопоточность.

**Q: Как UMSFA взаимодействует с облаком?**  
A: UMSFA создает локальный файл `.ms`. Вы можете загрузить его в облако вручную.

## 🔧 Использование в своем коде

### Простой пример

```python
from umsfa import MSArchiver

# Создание архива
archiver = MSArchiver(password="MyPassword")
archiver.archive("my_folder", "backup.ms")

# Распаковка
archiver.extract("backup.ms", "restored")

# Информация
info = archiver.get_archive_info("backup.ms")
print(f"Зашифрован: {info['encrypted']}")
```

### Функция резервного копирования

```python
from umsfa import MSArchiver
import os

def backup_with_timestamp(folder, password):
    from datetime import datetime
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    output = f"backup_{timestamp}.ms"
    
    archiver = MSArchiver(password=password)
    if archiver.archive(folder, output):
        size = os.path.getsize(output) / 1024 / 1024
        print(f"✅ Резервная копия: {output} ({size:.2f} МБ)")
        return output
    return None
```

## 📞 Поддержка и помощь

### Если что-то не работает:

1. **Проверьте требования:** Python 3.7+, все пакеты установлены
2. **Прочитайте FAQ:** см. раздел выше
3. **Посмотрите примеры:** см. `examples.py`
4. **Читайте логи:** программа выводит понятные сообщения об ошибках

### Типичные ошибки

| Ошибка | Решение |
|--------|---------|
| `ModuleNotFoundError: No module named 'cryptography'` | Запустите: `pip install -r requirements.txt` |
| `FileNotFoundError: [Errno 2] No such file or directory` | Проверьте путь к файлу/папке |
| `not a gzip file` при распаковке | Неверный пароль или архив повреждён |
| `PermissionError` | Проверьте права доступа к файлам |
